# 基质刷取词条筛选器
用于筛选明日方舟终末地中，武陵地区能量淤积点刷取的基质词条，便于针对特定武器刷取合适的基质
## 使用方法
* 直接下载并运行.exe文件，该exe文件会随着版本更新而更新
* 在python环境下运行main.py ，文件以list的形式记录了不同能量淤积点可以产出的基质词条，用户可自行添加
## 详细说明
* 双击运行.exe文件
* 在下拉框中选择想要刷出的附加属性和技能属性
* 点击“查找基质刷取点”按键，软件会自动筛选武陵地区符合要求的能量淤积点
